# Typography Direction

## Fonts
- Headings: Playfair Display — high contrast serif, editorial weight
- Body: Inter — clean, readable, modern sans
- Both available via next/font/google

## Scale (mobile first)
- Hero headline: 48px mobile / 80px desktop / 96px large
- Section headline: 36px mobile / 56px desktop
- Subheading: 20px mobile / 28px desktop
- Body: 16px mobile / 18px desktop
- Label/metadata: 12–14px, cadet (#8D99AE), letter-spaced

## Rules
- Headlines always Ghost White (#EDF2F4) on Space Cadet (#2B2D42)
- Never red for body text
- Generous line height on headlines: 1.1–1.2
- Body line height: 1.6–1.7
- All caps sparingly — labels and section tags only
- Letter spacing on all-caps labels: 0.08–0.12em

## Feeling
Magazine cover. Not app UI.
The typography should do most of the design work.
A headline set well needs no decoration around it.
